Responsive Navigation

This navigation bar automatically switches between desktop and mobile layouts. Resize your browser window to see the transformation!

On mobile devices, the navigation links are hidden behind a hamburger menu that slides in from the right when clicked. The overlay provides a dimmed background to focus on the navigation.

The design features modern elements like smooth animations, subtle shadows, and a glass-like effect on the navigation bar.
